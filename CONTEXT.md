# Contexto do Projeto: Site Sensen Games

## 1. Visão Geral do Projeto

O projeto consiste na criação de um site portfólio para um estúdio de desenvolvimento de jogos chamado "Sensen Games". O objetivo é apresentar os jogos do estúdio, fornecer informações sobre a equipe e facilitar o contato com a comunidade, imprensa e criadores de conteúdo.

## 2. Tecnologias Definidas

A stack de tecnologia para o desenvolvimento já foi definida e deve ser seguida:

- **Framework Principal:** React.js (utilizando Vite.js).
- **Estilização:** Tailwind CSS.
- **Roteamento:** React Router DOM.
- **Ícones:** React Icons.
- **Formulários:** React Hook Form.

## 3. Estrutura do Projeto

O projeto seguirá uma estrutura de pastas predefinida para organizar componentes, páginas, assets e dados. As principais pastas são:

- `src/components`: Para componentes reutilizáveis (Header, Footer, GameCard).
- `src/pages`: Para as páginas principais da aplicação (HomePage, CatalogPage, ContactPage).
- `src/data`: Para armazenar os dados dos jogos em um arquivo `games.js`.
- `src/assets`: Para imagens globais como o logo do estúdio.

## 4. Funcionalidades Chave

- **Catálogo de Jogos:** Uma página para exibir todos os jogos do estúdio. Os cards dos jogos devem mostrar uma imagem estática e, ao passar o mouse, exibir um vídeo de preview.
- **Detalhes do Jogo:** Um modal que abre ao clicar em um jogo, mostrando informações detalhadas.
- **Página Inicial:** Apresentará jogos em destaque e informações sobre o estúdio.
- **Formulários:** Formulário de contato e um formulário para inscrição em uma mailing list.
